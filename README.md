# Pulse Observability SDKs

Production-grade observability for mobile applications, powered by OpenTelemetry.

## SDKs

### [Pulse Android SDK](./pulse-android-otel)

OpenTelemetry-based observability SDK for native Android applications.

- Crash reporting and exception tracking
- Performance monitoring with distributed tracing
- Network request instrumentation
- ANR detection
- Activity/Fragment lifecycle tracking
- Slow render detection

**Requirements:** Android API 21+

[View Android SDK Documentation →](./pulse-android-otel)

---

### [Pulse React Native SDK](./pulse-react-native-otel)

Observability SDK for React Native applications with native Android integration.

- JavaScript crash and exception tracking
- Performance tracing
- Automatic network monitoring
- React Navigation integration
- Custom event tracking
- User identification

**Platform Support:** Android (API 24+) | iOS (Coming soon)

[View React Native SDK Documentation →](./pulse-react-native-otel)
