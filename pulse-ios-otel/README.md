# Pulse iOS SDK

Pulse iOS SDK is a production-grade observability SDK for iOS applications built on OpenTelemetry.

The SDK provides comprehensive Real User Monitoring (RUM) capabilities including error tracking, performance monitoring, network instrumentation, and distributed tracing.
