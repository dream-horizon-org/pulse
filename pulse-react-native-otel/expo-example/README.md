# Pulse Expo Example App

This is an example Expo app to test the Pulse React Native SDK Expo config plugin.

## Quick Start

1. **Install dependencies:**
   ```bash
   npm install
   ```

2. **Run prebuild:**
   ```bash
   npm run prebuild
   ```
   
   This generates native projects and applies the Pulse config plugin.

3. **Run the app:**
   ```bash
   npm run android
   # or
   npm run ios
   ```
